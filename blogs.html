Whenever you face data types error,just check the mapping sheet of data types.

while working with sqoop export from hadoop system to Teradata.We faced an data types error.For example we had a table in teradata with one of the column having a data type as Decimal(18,0) and same data types in hive. So while exporting this data we faced the data type mismatch error.So we checked the mapping sheet and found that teradata decimal was compatible with string.so we changed the data types to string and it worked...
